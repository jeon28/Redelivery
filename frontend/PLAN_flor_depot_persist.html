2026-05-21 · 사용자별 / region 별 마지막 선택 자동 복원.

1. 목표

  • FLOR 임대사 선택 시 매번 반납CY 를 묻지 않음.
  • (선사, 임대사, 반납지역) 별 마지막 선택을 자동 복원해 표시.
  • 변경하고 싶을 때만 사용자가 버튼 클릭 → 모달 (현 동작 유지).
  • 처음 사용 / 이력 없는 조합은 기존처럼 빨간 버튼 "미선택" 으로 안내.

사용자가 모달에서 새 depot 선택 화면에만 반영, 다음 로드 때 사라짐 (선사,임대사,region) 키 아래 history 에 저장 → 다음번 자동 복원
history 의 depot 이 백엔드 목록에서 사라진 경우 해당 사항 없음 모달 열 때 옵션 매칭 실패 → 자동으로 ''로 폴백 + 사용자에게 재선택 요구 (현 미선택 UI 그대로)

4. 저장 위치 결정

localStorage (per browser) 로 갑니다 — 기존 redelivery_ui_state_v1 패턴과 일관. "사용자 별로" 가 같은 브라우저를 쓰는 한 명의미라면 이걸로 충분. 서버에 사용자 단위로 동기화(여러 기기 공유)가 필요하면 별도 결정 후 backend 작업 추가.

6.1 헬퍼 함수 예시

const LS_DEPOT_KEY = 'redelivery_flor_depot_history_v1'

function loadDepotHistory(): Record<string, string> {
  if (typeof window === 'undefined') return {}
  try {
    return JSON.parse(localStorage.getItem(LS_DEPOT_KEY) ?? '{}')
  } catch {
    return {}
  }
}

function saveDepotHistory(key: string, label: string) {
  if (typeof window === 'undefined') return
  if (!label) return  // 빈 값은 저장 안 함
  try {
    const prev = loadDepotHistory()
    localStorage.setItem(LS_DEPOT_KEY, JSON.stringify({ ...prev, [key]: label }))
  } catch {}
}

function depotHistoryKey(company: string, lessor: string, region: string): string {
  return `${company}|${lessor}|${region}`
}

6.2 useEffect 교체

// 변경 전:
useEffect(() => {
    setDepotLabel('')
}, [company, region, isFlor])

// 변경 후:
useEffect(() => {
    if (!isFlor) {
        setDepotLabel('')
        return
    }
    const key = depotHistoryKey(company, lessor, region)
    const history = loadDepotHistory()
    setDepotLabel(history[key] ?? '')
}, [company, lessor, region, isFlor])   // ← lessor 의존성 추가

6.3 모달 onSelect 콜백

// 변경 전:
<DepotPickerModal
    ...
    onSelect={setDepotLabel}
/>

// 변경 후:
<DepotPickerModal
    ...
    onSelect={(label) => {
        setDepotLabel(label)
        const key = depotHistoryKey(company, lessor, region)
        saveDepotHistory(key, label)
    }}
/>

7. UI 표시 변화

버튼 자체는 현 디자인 그대로:

  • history 값 있음 → 회색 톤 + "🏭 (KRPUS03) Seyang Logistics ..." (기존 채워진 상태와 동일)
  • history 값 없음 → 빨강 톤 + "🏭 부산 반납CY 미선택 — 클릭하여 선택" (기존 미선택과 동일)

즉 추가 UI 없음. 변경된 건 초기 채워짐 여부만.

9. 비변경 / 위험

  • 백엔드 변경 없음.
  • 다른 임대사 (TEXA/TRIT/GOLD/GESE) — depot 선택 자체가 없어 무관.
